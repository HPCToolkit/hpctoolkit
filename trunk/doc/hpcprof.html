User Commands: hpcprof(1)

SYNOPSIS

hpcprof [options] executable hpcrun-file...

hpcprof -p [options] executable hpcrun-file...

Location: ${HPCTOOLKIT}/bin

DESCRIPTION

hpcprof is a utility for interpreting execution profiles (event-based program counter historgrams collected) with hpcrun. hpcprof has two basic modes: one to output a profile in a human-readable ASCII form, and another to output profiles in a machine-readable form (XML) for use with hpcview, which will correlate the profile data with program structure information (produced by the bloop binary analyzer) and source code.

ARGUMENTS

Options: General

-h, --help
Print help.
--debug
Print debugging information
-d, --directory dir
Search dir for source files.
-D, --recursive-directory dir
Search dir recursively for source files.
--force
Show data that is not accurate

Options: Text and HTML mode

-e, --everything
Show all information
-f, --files
Show all files
-r, --funcs
Show all functions
-l, --lines
Show all lines
-a, --annotate file
Annotate source file file with metrics
-n, --number
Show number of samples, not percent
-s, --show thres
Set threshhold to thres for showing aggregate data
-H, --html dir
Output HTML into directory dir.

Options: PROFILE mode

-p, --profile
Dump profile data in PROFILE format for consumption by hpcview

NOTES

hpcprof currently runs only on Linux.
