User Commands: hvprof(1)

SYNOPSIS

  1. Recompile your application, linking in vprof's vmon profiling library.
  2. Run your application.
  3. Convert vmon binary profiling information into PROFILE format using cprof.

cprof -p binary vmon_profile

cprof --hvprof_profile binary vmon_profile

Location: ${HPCTOOLKIT}/src/external_tools/hvprof (source code; not included in all distributions)

DESCRIPTION

hvprof is a modification of The Visual Profiler (by Curtis L. Janssen) a tool that uses PAPI to collect profiling information. We modified to the tool to package profile data into HPCTool's PROFILE format for input into hpcview. In addition, we have verified that PAPI profiling can successfully be performed on ia64-Linux platforms (Itanium and Itanium2).

ARGUMENTS

binary
The binary used to collect the profiling data.
vmon_profile
The file that contains binary profiling information. The default is vmon.out
-p or --hvprof_profile
Instruct cprof to convert the profiling data in vmon_profile to HPCTool's PROFILE format instead of to its default output. Either option name may be used, but other cprof regular option should be given.

NOTES

We have not tested on Pentium IV machines, though it should now be supported with PAPI 2.3.1 and greater.
