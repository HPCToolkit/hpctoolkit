System Prerequisites

[Contents]

  1. To reduce the effort needed to distribute HPCToolkit, we have opted to distribute it in source form only.

  2. HPCToolkit requires an ANSI/ISO C/C++ compliant compiler. Generally speaking any recent GCC (4.x, 3.3+) should be fine. Some vendor compilers that support GNU extensions also work. (GNU binutils employs GNU extensions.)

    • Linux-x86_64 (x86):
      • GCC C++/C 4.x, 3.4
      • Intel C++/C 10.x, 9.x
    • Linux-ia64:
      • GCC C++/C 4.x, 3.4
      • Intel C++/C 9.x
    • Linux-MIPS:
      • GCC C++/C 4.x
    • Linux-POWER:
      • GCC C++/C 4.x

  3. hpcrun, a tool for collecting performance data on Linux, requires an installation of PAPI 3.0 or greater.

  4. hpcviewer is a Rich Client application that can be used on a variety of platforms. It needs a Java 1.5 runtime environment (JRE) and gtk (on Linux).

  5. HPCToolkit requires a basic version of GNU make, not a vendor supplied make. We have successfully used versions 3.70, 3.76, 3.77, 3.78, 3.79 3.80.

Downloading the Sources

[Contents]

Download and extract the following:

  1. An HPCToolkit tarball

  2. An HPCToolkit externals tarball. This contains several external libraries, e.g., OpenAnalysis, Xerces-C++ and GNU binutils. In some cases it may be possible to use versions of these libraries that are already installed on your system. However, we do not support such configurations.

  3. An hpcviewer build.

To extract the tarball zoo.tar.gz, execute:

tar zxvf zoo.tar.gz

Building and Installing

[Contents]

The build and install follows autotools conventions (./configure && make && make install). In each case, more options are available by executing ./configure --help. See the README file for more details.

The typical build is as follows:

  1. Build and install HPCToolkit's externals, which typically does not require any special options:

    cd hpctoolkit-externals
    ./configure
    make all
    make distclean

  2. Build and install HPCToolkit:

    cd hpctoolkit
    ./configure --prefix=PATH --with-externals=PATH --with-papi=PATH
    make install

    This will build and install HPCToolkit in the paths given above. Because the installation is compact and self-contained, you may rename or move it to another location. All source code may be removed.
