Obtaining and Building HPCToolkit Software

Prerequisites & Instructions

Releases

  • For a complete installation, all four tarballs are necessary.
    hpctoolkit-5.1.0-release.tar.gz (source)
    hpctoolkit-externals-5.1.0-release.tar.gz (source)
    hpcviewer-5.1.0-release-<platform>.tgz (platform-specific build)
    hpctraceviewer-5.1.0-release-<platform>.tgz (platform-specific build)

Subversion Repository

  • To obtain a snapshot of the latest version of 'trunk':
    hpctoolkit (trunk) svn co https://outreach.scidac.gov/svn/hpctoolkit/trunk hpctoolkit
    hpctoolkit-externals svn co https://outreach.scidac.gov/svn/hpctoolkit/externals hpctoolkit-externals
    hpcviewer (trunk)
    hpctraceviewer (trunk)     		svn co https://outreach.scidac.gov/svn/hpcviewer/trunk hpcviewer
