Questions, Problems, Bugs

N.B.: When contacting us or submitting a bug report, please tell us about the following:

  • System. E.g., OS/architecture (uname -a), compiler (gcc -v), PAPI.
  • HPCToolkit. E.g., version; build information from <hpctoolkit-build>/config.log, which is generated after running configure.
  • Profiled application: E.g., language, whether dynamically or statically linked.
  • Relevant profiling/analysis commands. If you have used hpcrun, please include the results of executing:
    hpcsummary --all <hpctoolkit-measurements>
  • Email: hpc =at= cs.rice.edu
