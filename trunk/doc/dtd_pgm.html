File Format: PGM

SYNOPSIS

XML DTD (Document Type Definition): ${HPCTOOLKIT}/lib/dtd/PGM.dtd.

Suggested file extension: `.psxml'

PGM files specify program structure, i.e. they describe the source code structure of a program. By default, bloop outputs data in this format.

DESCRIPTION

The format of a PGM file should be fairly self-explanatory. It is essentially a program scope tree (PGM) with scopes for load modules (LM), files (F), procedures (P), loops (L) and statement ranges (S). Loops may be arbitrarily nested. The group (G) element allows one to specify and label aribrary sets of the above scopes (except PGM) for later reference.

EXAMPLES

The examples below are snippets of bloop output. Note that `n' attributes give names (such as file names, procedure names, label names), and `b' and `e' give an item's begin and end source code line numbers, respectively.
