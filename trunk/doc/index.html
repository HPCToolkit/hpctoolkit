Introduction

HPCToolkit is an open-source suite of multi-platform tools for profile-based performance analysis of applications. Please visit our web page for an overview of the toolset.

Building, Installing and Using

To download, build and install the toolkit, follow these instructions. Documentation and Examples can be found online.

Man pages are available locally (see top menu) and online.
