Obtaining and Building HPCToolkit Software

Prerequisites & Instructions

Source code for HPCToolkit

To build HPCToolkit from source, you need to download and build both hpctoolkit and hpctoolkit-externals.
    hpctoolkit svn co http://hpctoolkit.googlecode.com/svn/trunk hpctoolkit
    hpctoolkit-externals svn co http://hpctoolkit.googlecode.com/svn/externals hpctoolkit-externals

Binary releases for HPCToolkit GUI interfaces (recommended for tool users)

Due to the difficulty of building the hpcviewer and hpctraceviewer graphical user interfaces from source using Eclipse, we provide binary releases for these tools.
    hpcviewer-5.3.2-release-<platform>.tgz Select the appropriate binary releases for your target platforms of choice.
    hpctraceviewer-5.3.2-release-<platform>.tgz Select the appropriate binary releases for your target platforms of choice.

Source code for HPCToolkit GUI interfaces (recommended for tool developers only)

Source code for HPCToolkit's hpcviewer and hpctraceviewer graphical user interfaces is available from their subversion repositories. Unless you are a tool developer who wants to modify the capabilities of these graphical user interfaces, we recommend downloading the binary releases.
    hpcviewer
    (based on Eclipse RCP)     		svn co http://hpcviewer.googlecode.com/svn/trunk/edu.rice.cs.hpc.common
    svn co http://hpcviewer.googlecode.com/svn/trunk/edu.rice.cs.hpc.data
    svn co http://hpcviewer.googlecode.com/svn/trunk/edu.rice.cs.hpc.viewer
    hpctraceviewer
    (based on Eclipse RCP)     		svn co http://hpcviewer.googlecode.com/svn/trunk/edu.rice.cs.hpc.common
    svn co http://hpcviewer.googlecode.com/svn/trunk/edu.rice.cs.hpc.data
    svn co http://hpcviewer.googlecode.com/svn/trunk/edu.rice.cs.hpc.traceviewer
