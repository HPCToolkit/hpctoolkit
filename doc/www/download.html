Binary Releases of HPCToolkit's hpcviewer Graphical User Interface

Note: As of the 2020.12 release, hpcviewer supports trace views in addition to profile views, eliminating the need for a separate hpctraceviewer.

Requirements

  • On all platforms: Java 11 or newer
  • On Linux: GTK 3.20 or newer

Installing Java 11

The instructions to install Java:
  1. Download Java JDK
    • Recommended: spack install openjdk
    • Or download from:
      • AdoptOpenJDK
      • Azul Zulu (recommended for Mac M1 platform)
      • Oracle
  2. Set JAVA_HOME environment variable to the installed directory
Additional Mac OS directions if other Java versions are installed:
  • Leave all JDKs at their default location, under /Library/Java/JavaVirtualMachines. The system will pick the highest version by default.
  • To exclude a JDK from being picked by default, rename its Contents/Info.plist to: Info.plist.disabled.
    That JDK can still be used when $JAVA_HOME points to it, or explicitly referenced in a script or configuration. It will simply be ignored by system's java command.
