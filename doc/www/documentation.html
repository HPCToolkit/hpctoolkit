Questions, Problems, Bugs

Note: When contacting us or submitting a bug report using any of the means listed below, please tell us the following:

  1. System information: OS/architecture (uname -a), compiler (gcc -v), PAPI (version), Java (java -version).
  2. HPCToolkit: HPCToolkit version; if available, build information from <hpctoolkit-build>/config.log, which is generated after running configure.
  3. Profiled application: Information about your application, e.g., source language, whether dynamically or statically linked.
  4. Relevant profiling/analysis commands. If you have used hpcrun, please include the results of executing:
    hpcsummary --all <hpctoolkit-measurements>
